# Calendar4XD

一个 Windows 桌面日历日程应用原型，支持月历任务、单线程排程、任务完成、积分统计和任务线整体调整。

## 运行

```powershell
npm.cmd install
npm.cmd start
```

如果 PowerShell 提示 `npm.ps1` 被禁用，继续使用 `npm.cmd` 即可。

## 打包 Windows 便携版

```powershell
npm.cmd run pack:win
```

打包产物会输出到 `dist/`。

## 已实现功能

- 年/月切换和 42 格月历视图
- 每天显示任务、完成状态和积分
- 单线程模式开关
- 新增任务时自动顺延后续任务
- 任务提前完成时自动前移后续任务
- 任务编辑、删除、今日完成
- 总积分、本月积分、选中日积分和近 7 日柱状统计
- 整体任务线前移/后移

当前数据保存在本机浏览器存储中，适合作为第一版桌面原型；后续可以升级为文件数据库或 SQLite。
